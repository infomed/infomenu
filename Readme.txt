infomenu module 0.1
Originally By: Daniel Miguel (desarrollo@infomed.sld.cu)

June 22, 2009

DESCRIPTION

  * Allows to view drupal's menus in diferents ways and behaviors.
  
  * Allows to add customized blocks: 
      * Each block show a drupal menu in a specified way       
  * Allows to modify permisions

INSTRUCTIONS:

INSTALL

1) copy this to your modules directory

2) enable it at administer->build->modules

3) Add infomenu blocks at administer->infomenu->add

4) configure infomenu blocks at administer->infomenu->blocks

5) Add image to menu items at administer->build->menu->item->%->edit


UNINSTALL

1) disable it at administer->build->modules

2) delete all module's data at administer->build->modules->uninstall
